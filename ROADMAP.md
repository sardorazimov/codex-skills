# Roadmap

This roadmap describes near-term project direction. It is not a release promise.

## Milestone 1: Repository Foundation

- Establish Rust workspace boundaries.
- Establish Python package skeleton.
- Add contribution, security, and review templates.
- Add architecture and RFC documentation locations.

## Milestone 2: Protocol Shape

- Define initial protocol types.
- Document compatibility and validation rules.
- Add cross-crate integration tests.

## Milestone 3: Runtime Prototype

- Introduce minimal runtime orchestration.
- Add CLI commands for local smoke testing.
- Add Python examples that call supported public APIs.

## Milestone 4: Reliability and Developer Experience

- Add CI checks.
- Add benchmark harnesses for stable behavior.
- Improve contributor documentation based on real setup feedback.
